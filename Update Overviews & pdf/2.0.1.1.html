Game Center Version 2.0.1.1


**Release Date:** December 23, 2025


Update Summary


Version 2.0.1.1 is a hotfix release addressing critical issues found in 2.0.1. This patch provides essential stability improvements and fixes blocking issues for production deployment.


Bug Fixes


  • Fixed critical memory leak in score cache
  • Corrected JSON parsing error in GraphQL responses
  • Fixed UI crash when leaderboard has fewer than 10 entries
  • Patched socket timeout issues in API calls

    • Performance Improvements


  • Optimized GraphQL query performance (20% faster)
  • Reduced memory footprint by 15%
  • Improved startup time on slower connections

    • Changes


  • Updated error messages for better user clarity
  • Improved logging for debugging
  • Added retry logic for failed API requests

    • Stability Notes


  • All reported issues from 2.0.1 have been addressed
  • Recommended for production deployment
  • No breaking changes from 2.0.1
